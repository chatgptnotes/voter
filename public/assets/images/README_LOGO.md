# Logo Image Setup Instructions

## Required Logo Files

Please save your logo images in this folder with the following names:

### 1. Main Logo (Required)
**Filename:** `pop-logo.png`
- This is your main logo with "P•O•P" and "pulse of people" text
- Recommended size: 360x120px or similar aspect ratio
- Format: PNG with transparent background preferred

### 2. Icon Version (Optional but Recommended)
**Filename:** `pop-logo-icon.png`
- Just the "P•O•P" part without the text below
- Square format recommended
- Size: 100x100px or 200x200px
- Used for small spaces and mobile views

### 3. Compact Version (Optional)
**Filename:** `pop-logo-compact.png`
- Smaller horizontal version for mobile headers
- Recommended size: 150x50px

### 4. Favicon (Required)
**Filename:** `pop-favicon.png`
- Square format, just the icon or "POP" letters
- Size: 32x32px or 64x64px (will be resized automatically)
- Used as browser tab icon

## How to Save Your Logo

1. Save the logo image you provided as `pop-logo.png` in this folder:
   `D:\open project\pop\voter\public\assets\images\`

2. If you have different versions, save them with the names above

3. If you only have one logo file, save it as:
   - `pop-logo.png` (main usage)
   - Also save a copy as `pop-favicon.png` (for browser tab)

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG (if no transparency needed)
- SVG (if you have vector format)

## After Saving Images
The logo will automatically appear throughout the application. The Logo component has fallback text rendering if images are not found.

## Current Logo Colors (for reference)
- Purple: #9333EA (first P and O)
- Blue: #3B82F6 (last P)

- Gradient line: Purple to Blue
